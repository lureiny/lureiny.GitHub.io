环境配置

1、运行MacOS系统的电脑

2、Node.js

用来生成静态页面。Node.js可以去官网下载pkg包，直接安装即可。

3、Git

用来提交本地的文件到GitHub。Xcode自带Git。Xcode可以去App Store下载。如果没有安装Xcode，可以通过Homebrew下载安装。
