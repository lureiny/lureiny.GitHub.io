Docker使用记录

容器

创建新的容器：

docker run -i -t -d –name=docker_name ubuntu:16.04 /bin/bash

i：标准输入

t：创建虚拟终端

name：指定docker名字，可以不指定，不指定时系统会随机生成一个name

ubuntu：指定基于的版本

/bin/bash：指定运行的程序

查看创建的容器情况：

docker ps -a

列出容器情况，带“-a”为查看所有容易（包括已经停止的容器），否则为查看活动容器的情况

启动或者停止容器：

docker start/stop/kill id/name

删除容器：

docker rm id/name 不能删除正在运行的容器

进入容器：

docker attach id/name

镜像

下载镜像：

docker pull name

查看本地镜像：

docker images

nvidia-docker命令同docker，只不过是创建支持显卡的docker
